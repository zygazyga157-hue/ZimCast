ZIMCAST UI REDESIGN SPEC (ANALYTICS PAGE)

-----------------------------------
ANALYTICS PAGE DESIGN (STRUCTURE ONLY)
-----------------------------------

CORE PRINCIPLE:
Analytics should feel like a smart dashboard, not raw data blocks.

STRUCTURE:

1. HEADER
- Title: Analytics
- Subtitle (contextual description)
- Optional action buttons (e.g., Profile, Browse)

-----------------------------------

2. QUICK STATS GRID

Layout:
2x2 grid cards

Cards:
- Watch Time
- Engagement
- Favorite Category
- Matches / Sessions

Behavior:
- Equal sized cards
- Minimal text
- Focus on key metric
- Tappable for deeper insights (future)

-----------------------------------

3. ACTIVITY HEATMAP

Purpose:
Show user behavior patterns over time

Layout:
- Grid (days vs hours)
- Labels:
  - Days (vertical)
  - Time (horizontal)

Behavior:
- Cells represent activity intensity
- Hover/tap can reveal exact values (future)

-----------------------------------

4. CATEGORY BREAKDOWN

Purpose:
Show content consumption distribution

Layout:
- Section title
- List or progress bars per category

Example:
Music — duration + percentage

Behavior:
- Sorted by highest usage
- Expandable for more categories

-----------------------------------

5. INSIGHTS SECTION

Purpose:
Translate data into meaningful observations

Layout:
- Title: Insights
- Bullet list format

Content:
- Viewing habits
- Peak usage times
- Suggestions

Behavior:
- Dynamically generated (future)
- Easy to scan

-----------------------------------

6. TOP PROGRAMS

Purpose:
Highlight most consumed content

Layout:
Vertical list:

Each item:
- Rank (#1, #2, etc.)
- Program name
- Time watched
- Duration badge (right aligned)

Behavior:
- Sorted descending by watch time
- Tappable for details

-----------------------------------

7. RECENT ACTIVITY

Purpose:
Show latest user interactions

Layout:
Vertical list:

Each item:
- Program name
- Timestamp
- Category tag
- Duration

Behavior:
- Chronological order
- Compact card design

-----------------------------------

8. PAGE FLOW

1. User sees key stats immediately
2. Scrolls into behavior patterns (heatmap)
3. Understands preferences (categories)
4. Gets smart insights
5. Sees top and recent content

-----------------------------------

9. COMPONENTS

- StatCard
- HeatmapGrid
- CategoryBar
- InsightList
- RankedListItem
- ActivityItem

-----------------------------------

END OF SPEC
-----------------------------------
